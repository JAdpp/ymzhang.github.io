# Yangming Zhang Academic Homepage

Jekyll academic homepage for GitHub Pages.

Live site: https://jadpp.github.io/ymzhang.github.io/

Repository: https://github.com/JAdpp/ymzhang.github.io

## Local build

This machine may need Ruby first. When Ruby and Bundler are available:

```powershell
bundle install
bundle exec jekyll serve
```

## Deployment

Deploy from the `main` branch of `JAdpp/ymzhang.github.io` using GitHub Pages. The custom domain `ymzhang.space` is intentionally not configured in v1.

## Public CV

The public CV is generated from `scripts/generate_public_cv.py` and excludes phone number, direct email, and application-specific wording.
